# Privacy Checklist

Usar esta lista antes de guardar o publicar cualquier screenshot del proyecto.

## No mostrar

- nombre de usuario de Windows
- rutas locales completas
- barra de tareas, reloj o notificaciones
- pestanas del navegador con informacion personal
- nombres reales, DNI, CBU, alias o referencias bancarias reales
- saldos reales, movimientos reales o identificadores reales
- mails, chats o cualquier app abierta fuera del proyecto

## Si la captura es de Excel

- usar solo `FINANZAS_demo.xlsx`
- revisar que no haya hojas ocultas visibles por accidente
- mostrar solo la zona relevante de la tabla
- evitar encabezados con rutas o nombres personales

## Si la captura es bancaria

- usar solo imagenes demo o anonimizadas
- no dejar capturas reales en `docs/screenshots/`
- recortar mas de lo necesario antes de publicar
- revisar que no queden nombre del titular, saldo, banco, cuenta o fecha sensible si no aporta
- si hay duda, tapar o reemplazar el dato

## Antes de publicar

- abrir la imagen final y revisar al 100%
- confirmar que el archivo tenga nombre claro y ordenado
- guardar la captura en `docs/screenshots/`
